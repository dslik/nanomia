# Nanomia Adjustible CV Output

A simple 2HP CV level output module.

Initial features include:
- Adjustable ranges:
   - 0V - 5V
   - 0V - 12V
   - -12V - 5V
   - -12V - 12V
- Fine and coarse sliders
